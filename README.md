# Java Banking System

A Java-based banking system simulating basic banking operations like account management, transactions, and authentication.

## 🧰 Features
- User authentication
- Create accounts
- Deposit and withdraw funds
- Check balances
- View transaction history

## 🖥️ Technologies Used
- Java
- Java Swing (GUI)
- File-based data storage
